# 🚀 90-Day Web Development Challenge

This repository documents my **90-Day Challenge to strengthen my foundation in web development** using **HTML, CSS, and JavaScript**.

The purpose of this challenge is to develop **discipline, consistency, and deeper understanding of web technologies** through structured learning and hands-on coding practice.

As a **Computer Engineering graduate focusing on software development**, I created this challenge to improve my **technical skills, coding habits, and readiness for a career in the tech industry**.

# 📚 Learning Focus

This challenge focuses on three main areas of web development:

* **HTML** – structuring web pages
* **CSS** – styling and responsive layouts
* **JavaScript** – programming logic and interactivity

Each section in this repository contains **topic-based practice files and exercises** to reinforce my understanding of these concepts.
