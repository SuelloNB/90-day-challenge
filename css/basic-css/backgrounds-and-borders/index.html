Working with Backgrounds and Borders

Backround Image Property


/*Background Image Property*/
.bg-img{
    background-image: url("../assets/images/sunflower.png"); /*Set a background image to a container*/
}

Background Size Property

Contain Value


/*Contain Value*/
.bg-img-contain{
    background-image: url("../assets/images/sunflower.png"); /*Set a background image to a container*/
background-size: contain; /*set the image size to contain*/
}

Cover Value


/*Cover Value*/
.bg-img-cover{
    background-image: url("../assets/images/sunflower.png");
    background-size: cover; /*Sets the image size to cover*/
}

Background Repeat Property

No Repeat Value


/*No Repeat Value*/
.bg-img-no-repeat{
    background-image: url("../assets/images/sunflower.png");
    background-size: contain; /*Background size set to contain*/
    background-repeat: no-repeat; /*Set the background-repeat property to no-repeat so that the image will no longer repeat on the screen*/
}

Repeat-X Value


/*Repeat-x value*/
.bg-img-repeat-x{
    background-image: url("../assets/images/sunflower.png");
    background-size: contain; /*Background size set to contain*/
    background-repeat: repeat-x; /*Repeats the background image horizontally*/
}

Repeat-Y Value


/*Repeat-y value*/
.bg-img-repeat-y{
    background-image: url("../assets/images/sunflower.png");
    background-size: contain; /*Background size set to contain*/
    background-repeat: repeat-y; /*Repeats the background image vertically*/
}

Background Position Property

Center Top Value


/*Center top value*/
.bg-img-center-top{
    background-image: url("../assets/images/sunflower.png");
    background-size: contain; /*Sets the background size to contain*/
    background-repeat: no-repeat; /*Set the background-repeat property to no-repeat so that the image will no longer repeat on the screen*/
    background-position: center top; /*Sets the background positon to center top*/
}

As for the background position property, you can use this following value:

  • Top
  • Bottom
  • Left
  • Right
  • Center

Background Attachment Property

Fixed Value


/*Background attachment property*/
/*Fixed Value*/
.bg-img-fixed{
    background-image: url("../assets/images/sunflower.png");
    background-repeat: no-repeat; /*Sets the background's behavior to no-repeat*/
    background-position:  center top; /*Sets the images' position to center top*/
    background-attachment: fixed; /*Sets the background attachment property to fixed, where the background image stays in the same position on the screen*/
}

Short-hand Background Property


/*Short-hand background property*/
.bg-img-short-hand{
    background: no-repeat center top fixed url("../assets/images/sunflower.png"); /*Sets the background-position to center top, background-attachment to fixed and background-image directory*/
}

Background Gradient and It's Functions


/*Linear gradient*/
.linear-gradient{
    background: linear-gradient(to right, red, yellow); /*linear gradient that transitions from red to on the left to yellow on the right*/
    height: 25vh;
}    
        

/*Radial gradient*/
.radial-gradient{
    background: radial-gradient(circle closest-corner at center, red, yellow 50%, green); /*Creates a circular radial gradient centered in the element. Starts with red at the center, transitions to yellow at 50% of the radius, and ends with green*/
    height: 25vh;
}
