Working with Backgrounds and Borders

Backround Image Property


/*Background Image Property*/
.bg-img{
    background-image: url("../assets/images/sunflower.png"); /*Set a background image to a container*/
}

Background Size Property

Contain Value


/*Contain Value*/
.bg-img-size{
    background-image: url("../assets/images/sunflower.png"); /*Set a background image to a container*/
background-size: contain; /*set the image size to contain*/
}

Cover Value


/*Cover Value*/
.bg-img-cover{
    background-image: url("../assets/images/sunflower.png");
    background-size: cover; /*Sets the image size to cover*/
}
