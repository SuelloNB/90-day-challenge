Link Styles

Default Link Styles

Example link 
/*Default link styles*/
.default-link-styles a{
    color: blue; /*Color is unnecessary since the default color of an anchor element is blue. This is just an example.*/
}
.default-link-styles a:visited{
    color: purple; /*Color for visited is unnecessary since the default color for visited anchor element is purple. This is just an example.*/
}

Hover and Active Psuedo Classes

Link 1 

/*Hover and Active Psuedo Classes*/
.hover-and-active-psuedo-classes a:hover{
    color: red; /*Sets the text color to red when hovered*/
}
                
.hover-and-active-psuedo-classes a:active{
    color: black;   /*Sets the text color to black when clicked*/
}
